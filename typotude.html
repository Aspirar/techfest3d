Description

  • Game type: individual ( only one player can participate from one user id,it’s not a team game)
  • The game is divided into two phases. In the first phase the player has to solve basic aptitude questions.
  • Number of questions solved would act as a fuel which is used for driving car in the second phase of the game. I.e. More the number of questions solved more will be the quantity of fuel the player will get to drive the car.
  • The player can leave the first phase any time. But after leaving this phase he/she cannot return back to this phase to solve more problems.
  • In the second phase player needs to type the words displayed on the screen. Typing speed of the player corresponds to the driving speed of the car.
  • In a particular time interval, fuel tank keeps on decreasing. The player has to type as many words as possible before his/her car’s fuel tank gets empty.
  • There will be penalty for wrong typed words.
  • Winner will be decided on the basis of accuracy, number of words typed and taking into account penalties due to mistyped words.
